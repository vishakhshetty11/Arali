Customer Management System (MERN Stack)
This is a full-stack Customer Management System built using React (Vite), Node.js, Express, and MongoDB.
________________________________________
🚀 Live Links
Frontend (Vercel): https://customer-management-six.vercel.app
Backend (Render): https://arali-customer-backend.onrender.com
________________________________________
🛠️ Tech Stack
•	Frontend: React (Vite)
•	Backend: Node.js, Express
•	Deployment: Vercel + Render
________________________________________
📁 Project Structure
/frontend   → React app
/backend    → Express API
________________________________________
⚙️ Installation
1. Clone repository
git clone https://github.com/vishakhshetty11/Arali.git
cd Arali
________________________________________
2. Frontend setup
cd frontend
npm install
npm run dev
________________________________________
3. Backend setup
cd backend
npm install
npm start
________________________________________
🔌 API Endpoints
Customers
•	GET /customers → Get all customers
•	POST /customers → Add customer
•	DELETE /customers/:id → Delete customer
________________________________________
🌐 Environment Variables
Backend (.env)
PORT=5000
________________________________________
🚀 Deployment
Frontend (Vercel)
•	Root directory: frontend
•	Build command: npm run build
•	Output directory: dist
Backend (Render)
•	Start command: npm start
